# Cedar

> The best operating system hasn't been made yet.

A pre-product landing page for Cedar OS — a Linux-based OS built around one idea: a computer should disappear.

## Stack

- [Next.js 14](https://nextjs.org/) — App Router
- [Poppins](https://fonts.google.com/specimen/Poppins) — via `next/font/google`
- Deployed on [Vercel](https://vercel.com/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

## Deploy

Push to GitHub and connect the repo to Vercel. Zero configuration needed.
